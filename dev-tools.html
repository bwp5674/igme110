Tools for Developers

Looking to develop your own Alexa Skill, Google Home Action, or a new voice platform entirely? Here are some tools you can use:

  • Google DialogFlow - an end-to-end, build-once deploy-everywhere development suite for creating conversational interfaces
  • wit.ai - Natural Language for Developers
  • AWS SDK Code Generator - for Alexa Skill Language Models
  • Amazon Lex - Conversational interfaces for your applications powered by the same deep learning technologies as Alexa
  • echosim.io - Alexa Skill Testing Tool
